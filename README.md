# Chinny's Resume

## Requirements

* Pandoc [https://pandoc.org/installing.html]
* wkhtmltopdf [https://wkhtmltopdf.org/downloads.html]

## Usage

* Make edits to jchin_resume.html
* Run md-to-html.ps1 script to convert to HTML
* Run html-to-pdf.ps1 script to convert to PDF
